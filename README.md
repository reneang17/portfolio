# René Ángeles — Personal Project Portfolio

This repository hosts the source for my personal portfolio website and showcases selected projects I’ve worked on. The site is a simple, static portfolio that highlights projects, tech stacks, and links to demos and repositories.

## Projects

- Author Encoder
  - Description: A book recommendation concept from my AI fellowship at Insight Data Science that encodes writing style using NLP to find similar authors.
  - Tech: PyTorch, AWS SageMaker, AWS Lambda, Docker
  - Repo: https://github.com/reneang17/authorencoder

- PDF Browser Reader
  - Description: A frontend built on PDF.js to improve the browser PDF reading experience.
  - Tech: HTML, CSS, JavaScript, PDF.js
  - Demo: https://reneang17.github.io/pdf-viewer-frontend/
  - Repo: https://github.com/reneang17/pdf-viewer-frontend

- Fullstack PDF Browser Reader
  - Description: A full-stack extension of the PDF reader that functions as a virtual library with an API for file storage and management.
  - Tech: Node + Express, PostgreSQL + S3 (Supabase), React (frontend with PDF.js), Vercel
  - Backend Repo: https://github.com/reneang17/pdf-storage-api
  - Frontend Repo: https://github.com/reneang17/pdf-viewer-poc
